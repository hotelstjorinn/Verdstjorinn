Cloudbeds — API lykill
⚠️ Takmarkaður stuðningur beint úr vafra. Cloudbeds API notar OAuth 2.0 sem krefst backend-þjóns vegna CORS-takmarkana. Ráðlögð leið: nota Útflutning (CSV/JSON) og hlaða upp handvirkt, eða setja upp einfaldan Node.js proxy.
Fáðu API lykil úr: Manage → Apps & Marketplace → API Credentials
Node.js Proxy — uppsetning
Einfaldasta leiðin til að nota Cloudbeds API er að setja upp þennan proxy á tölvunni þinni (eða VPS):
# 1. Settu upp Node.js og keyrðu þetta
npm init -y
npm install express cors node-fetch

# 2. Búðu til proxy.js með þessum kóða og keyrðu: node proxy.js
const express = require('express');
const cors = require('cors');
const app = express();
app.use(cors()); app.use(express.json());
app.post('/cb', async (req, res) => {
  const r = await fetch('https://hotels.cloudbeds.com/api/v1.2/'+req.body.method, {
    method: req.body.httpMethod||'GET',
    headers: {Authorization: 'Bearer '+req.body.token},
    body: req.body.httpMethod==='POST' ? new URLSearchParams(req.body.params) : undefined
  });
  res.json(await r.json());
});
app.listen(3001);
Þegar proxy er keyrt á port 3001 mun forritið nota http://localhost:3001/cb sem millilið
